# Projet Pokedex

Le projet Pokedex est une application web conçue pour les amateurs de Pokémon. Il permet aux utilisateurs de rechercher et d'explorer différentes informations sur les Pokémon, tels que leurs caractéristiques, leurs types, leurs évolutions et bien plus encore.

## Fonctionnalités

- Recherche de Pokémon : Les utilisateurs peuvent rechercher des Pokémon spécifiques en utilisant leur nom.
- Détails des Pokémon : Une fois qu'un Pokémon est sélectionné, les utilisateurs peuvent accéder à des informations détaillées telles que les statistiques, et les capacités.
- Galerie d'images : Les utilisateurs peuvent visualiser des images haute résolution des Pokémon pour mieux les connaître.

## Technologies utilisées

- HTML, CSS et JavaScript pour la structure, le style et l'interactivité de l'application.
- API Pokémon : L'application utilise une API de "pokeapi" pour récupérer les données sur les Pokémon.
